Classificação Qualis

Este aplicativo permite que você classifique artigos de um CV Lattes com base em dados Qualis. Para usar, siga as seguintes etapas:

  1. Faça upload de um arquivo XML com o CV Lattes. Para obter o CV no formato XML, navegue até seu Lattes e clique no botão `XML` no canto superior direito da página.
  2. Após o upload do arquivo XML, clique em "Enviar".
  3. O aplicativo irá processar o arquivo e classificar os artigos com base nos dados Qualis.
  4. Os resultados serão exibidos na caixa de texto abaixo do botão de envio. Você pode copiar os resultados para a área de transferência clicando em "Copiar para a área de transferência".

Para contribuições e reportar problemas, por favor, acesse nosso repositório no GitHub.
